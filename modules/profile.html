PersonaIdentifiers

PersonaIdentifiers: { AadObjectId?: string; Smtp?: string; Upn?: string }
  • Defined in packages/teams-js/dist/MicrosoftTeams.d.ts:5164

The set of identifiers that are supported for resolving the persona.

At least one is required, and if multiple are provided then only the highest priority one will be used (AadObjectId > Upn > Smtp).

Type declaration

  • Optional Readonly AadObjectId?: string

    The object id in Azure Active Directory.

    This id is guaranteed to be unique for an object within a tenant, and so if provided will lead to a more performant lookup. It can be resolved via MS Graph (see https://docs.microsoft.com/en-us/graph/api/resources/users for examples).

  • Optional Readonly Smtp?: string

    The primary SMTP address.

  • Optional Readonly Upn?: string

    The user principle name.

TriggerType

TriggerType: "MouseHover" | "MouseClick" | "KeyboardPress" | "AppRequest"
  • Defined in packages/teams-js/dist/MicrosoftTeams.d.ts:5157

The type of the profile trigger.

  • MouseHover: The user hovered a target.
  • MouseClick: The user clicked a target.
  • KeyboardPress: The user initiated the show profile request with their keyboard.
  • AppRequest: The show profile request is happening programmatically, without direct user interaction.
